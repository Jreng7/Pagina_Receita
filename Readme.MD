# Página de Receita - Projeto do Curso FullStack na Rocketseat

Estou iniciando meus estudos no curso FullStack da Rocketseat, e durante essa jornada, vou desenvolver um projeto chamado **"Página de Receita"**. Nesse projeto, aplicarei as tecnologias que estou aprendendo, como:

- **Git**: Controle de versão e gerenciamento do código do projeto.
- **HTML**: Estruturação da página de receitas com uma marcação semântica clara.
- **CSS**: Estilização da página para garantir um layout moderno e responsivo.
- **JavaScript**: Implementação de interatividade e lógica no front-end para melhorar a experiência do usuário.

Além disso, esse projeto me ajudará a consolidar o aprendizado e construir um portfólio sólido.
